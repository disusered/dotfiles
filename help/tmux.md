tmux
---------

|   | **Movement**    |
|---|-----------------|
| } | Move pane right |
| { | Move pane left  |

|           | **Normal Mode**              |
|-----------|------------------------------|
| ^y        | Copy current command in pane |
| ^Y        | Copy PWD of pane             |
| ^, ctrl-g | Find git status files        |
| ^, ctrl-i | Find ip addresses            |
| ^, ctrl-u | Find url addresses           |

|    | **Copy Mode**          |
|----|------------------------|
| ^[ | Enter copy mode        |
| v  | Begin text selection   |
| y  | Copy text              |
| Y  | Put text               |
| ?  | Search up              |
| /  | Search down            |
| n  | Jump to next match     |
| N  | Jump to previous match |

|    | **Commands**   |
|----|----------------|
| ^$ | Rename session |
| ^A | Rename session |
