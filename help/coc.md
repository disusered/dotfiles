---------

| K   | Show docs             |
| gd  | Go to definition      |
| gD  | Go to declaration     |
| gy  | Go to type definition |
| gi  | Go to implementations |

| mr  | Rename symbol         |
| ma  | CodeAction line       |
| mas | CodeAction selected   |
