coc.nvim
---------

|    | **Action**                  |
|----|-----------------------------|
| ^l | Coc List                    |
| K  | Show docs                   |
| gd | Go to definition            |
| gD | Go to declaration           |
| gy | Go to type definition       |
| gi | Go to implementations       |
| gr | Show references             |
| mr | Rename symbol               |
| ma | CodeAction line or selected |
| mq | QuickFix line or selected   |
| ]g | Go to next diagnostic       |
| [g | Go to previous diagnostic   |
